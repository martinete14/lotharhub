# Lothar Hub

Interfaz web interactiva desarrollada con **HTML, CSS y JavaScript** como proyecto académico para la asignatura **Lenguajes de Marcas y Sistemas de Gestión de Información** del Grado Superior en Desarrollo de Aplicaciones Web.

La página funciona como un hub de comunidades gaming centrado en **MDQ Counter-Strike**, presentando contenido informativo, multimedia y funcionalidades interactivas.

---

## 📌 Objetivo del proyecto

Diseñar y desarrollar una página web moderna aplicando:

- Estructura HTML semántica
- Técnicas de maquetación con CSS
- Interactividad con JavaScript
- Buenas prácticas de SEO

---

## 🚀 Características

- Estructura HTML semántica (`header`, `nav`, `main`, `section`, `footer`)
- Diseño moderno con **Flexbox y CSS Grid**
- **Video de fondo** en la sección principal (Hero)
- Sección de gameplay con contenido multimedia
- Galería multimedia con videos
- Interactividad con **JavaScript y Event Listeners**
- Botón para copiar la IP del servidor
- Navegación suave entre secciones (**smooth scroll**)
- Animaciones CSS
- Diseño **responsive** con media queries
- Uso de sombras, bordes redondeados y tipografía web

---

## 🛠 Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts**

## 🎮 Sobre el contenido

El proyecto presenta información sobre la comunidad **MDQ Counter-Strike**, una comunidad histórica de servidores del juego con una larga trayectoria en Argentina y Sudamérica.

También incluye una sección secundaria dedicada a **TITAN Community**, una comunidad orientada a Minecraft y otros videojuegos.
